# AI Medical Research Search System

## Project Overview
This project is an AI-powered medical research paper search system that uses Sentence Transformers and FAISS for semantic search. It helps users find the most relevant medical research abstracts based on their search query.

## Features
- Semantic search using Sentence Transformers
- Fast similarity search with FAISS
- Best Matching Paper Highlight
- Search History Tracking
- Performance Evaluation
- Multiple Query Testing

## Technologies Used
- Python
- Google Colab
- FAISS
- Sentence Transformers
- Pandas
- NumPy

## Dataset
Medical research abstracts dataset containing paper titles and categories.

## How to Run
1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Run all cells.
4. Search using queries like:
   - covid vaccine
   - cancer
   - diabetes
   - brain tumor
